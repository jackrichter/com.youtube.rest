This is how to create a Java Restful Service tutorial on youtube

Part #1

Intro to this series.

Part #2

Start creating our Java Rest Service using Jersey Framework

For more information on Jersey, please go to http://jersey.java.net/
The series uses version 1.17, but this code uses version 1.18 (under /Oracle)

Two API resources were created in part 2:
http://com.youtube.rest/api/v1/status
http://com.youtube.rest/api/v1/status/version

Part # 3

We got to the Weblogic Server Admin Console
http://localhost:7001/console

There, at data source, we created a JDBC data source connection to our database with the JNDI name 308tubeOracle

Next we created a dedicated static class in it's own package Oracle308tube to have a method that returned us a Data Access
Object, to get a connection to our database.

Finally, we improved our api by introducing a method to check the status of our database
by returning Date/Time info in a read-only (GET) manner.

Part #4

Two new packages have been added (inventory and util). Inventory will allow you to manage computer parts in the PC_PARTS table. In the util package, a ToJSON class has been added to convert database resultset into JSON array.

One API resources was created in part 4:
http://localhost:7001/com.youtube.rest/api/v1/inventory - This url will return all the rows PC_PARTS table.

Part #5

We increase the complexity of this project by making the dao package hold all of our SQL and database code. We also created V2_inventory to replace V1_inventory resources. V2 is a improvement on V1. We created three new resources in episode 5, two API resources do the same thing but in different ways. The third API resource refines the search to pin point a specific product item.

Two of the API resources are for searching on the brand but but done differently:
The first one is by @QueryParam:
http://localhost:7001/com.youtube.rest/api/v2/inventory?brand=ASUS
The second one is by @PathParam:
http://localhost:7001/com.youtube.rest/api/v2/inventory/ASUS

The third api resource search by brand and item number using @PathParam:
http://localhost:7001/com.youtube.rest/api/v2/inventory/ASUS/168131318
